# Portfolio Website (React + CSS)

A beginner-friendly portfolio starter built with React and Vite.

## 1) Run locally

```bash
npm install
npm run dev
```

Then open the URL Vite shows in the terminal (usually `http://localhost:5173`).

## 2) Edit your content

Update the text in `src/App.jsx`:

- Change `Your Name`
- Replace project cards with your real projects
- Update email and GitHub links

Style changes go in:

- `src/App.css`
- `src/index.css`

## 3) Git workflow checkpoints

Use this flow as you build:

### Checkpoint A - Initial setup

After app creation and first working version:

```bash
git add .
git commit -m "Set up React portfolio starter with basic sections"
git push -u origin main
```

### Checkpoint B - Content update

After you replace placeholder text with your real info:

```bash
git add .
git commit -m "Add real portfolio content and contact details"
git push
```

### Checkpoint C - Style polish

After improving colors, spacing, images, and responsiveness:

```bash
git add .
git commit -m "Polish portfolio styling and responsive layout"
git push
```
